Sobre

Minha base sólida em tecnologias essenciais, como HTML, CSS e JavaScript, permite-me criar interfaces atraentes e responsivas que cativam os usuários desde o primeiro clique. Tenho experiência sólida no desenvolvimento com o ecossistema React, onde transformo ideias criativas em aplicativos web modernos e dinâmicos, incorporando também o TypeScript para garantir código robusto e escalável.

No lado do servidor, sou um entusiasta do Node.js e Express, e minha habilidade com o TypeORM e PostgreSQL facilita a construção de bancos de dados eficientes e gerenciáveis para aplicativos web de alta qualidade. Além disso, sou versátil na programação, com conhecimento em Python e Django, o que me permite oferecer soluções completas e personalizadas para qualquer projeto que assumo.

imdFlix

Esse foi um projeto realizado em grupo, o objetivo do projeto é solucionar o problema da falta de um serviço de streaming voltado para pessoas que não sabem o que querem assistir. Então na nossa aplicação o usuário pode ver análise de outros usuários, assistir ao trailer de um filme/série, deixar sua avaliação, além de poder realizar o cadastro, login, filtrar por categoria e por nome.

Principais tecnologias: React | TypeScript | React Hooks | Styled-Components | JSON Server | Git | Scrum | Trello | Figma

Motors Shop

O objetivo do projeto é criar uma concessionária virtual, conectando o usuário comprador com o usuário vendedor. Para esse projeto desenvolvemos uma aplicação com a possibilidade do usuário criar sua conta como comprador ou vendedor. O vendedor pode criar anúncios, adicionando o valor do produto, imagens, descrição, e muito mais. O comprador pode entrar em contato com o vendedor através do WhatsApp, comentar em algum anúncio e muito mais. Além disso, o projeto conta com funcionalidades de recuperação de senha, opção de excluir a conta, login e logout, e muito mais.

Principais tecnologias utilizadas no projeto: React | TypeScript | Styled-components | ReactRouterDOM | React-hook-form | Axios | React-Icons | Chakra-UI | Yup | Git | Jira | Figma
